# TCP (Transmission Control Protocol)

전송을 제어하는 프로토콜

**TCP의 특징**

- 연결 지향적 프로토콜.
- 높은 신뢰성 보장.
- 흐름 제어 및 혼잡 제어.
- UDP보다 속도 낮음.

예시: 웹 브라우저, 이메일, 파일 전송 등

# UDP (User Datagram Protocol)

데이터를 데이터그램 단위로 처리하는 프로토콜

**UDP의 특징**

- 비연결 지향적 프로토콜.
- 데이터 전송 시 오류 검사 및 복구 기능 X
- 단순하고 빠른 데이터 전송
- 낮은 신뢰성.

예시: 비디오 스트리밍, 등 실시간 응용 프로그램
